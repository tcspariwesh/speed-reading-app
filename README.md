# Speed Reading App1

A small React demo for speed reading.

## Features
- Paste text into a box
- Adjust speed with a slider
- Start, pause, and reset playback
- Shows one word at a time
- Displays a simple picture-like hint for the current word

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
merge start and pause buttons
